Model Health Dashboard

The Model Health Dashboard provides a real-time health score for your Revit model by analyzing key quality metrics.

Metrics Analyzed

  • Warnings — Total Revit warnings in the model
  • In-Place Families — Count of in-place families (performance impact)
  • Groups — Number of model/detail groups
  • CAD Imports — Imported DWG/DXF files
  • Redundant Rooms — Rooms with area = 0
  • Orphaned Views — Views not placed on sheets
  • Revit Links — Number of linked models
  • Unused Filters — Filters not applied to any view

Health Score

Score > 80 = Excellent | 50-80 = Attention Needed | < 50 = Critical

Usage

  1. Click Model Health in the QC Panel
  2. The dashboard scans the model automatically
  3. Click Refresh to re-scan after making changes
