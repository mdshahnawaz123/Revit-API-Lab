Room Wizard

Export elements belonging to host or linked rooms into a CSV file, easily modify their parameter data in Excel, and quickly import those changes directly back into Revit.

Features

  • Element Detection — Analyzes the geometric bounds of rooms (host or linked) to accurately find all architectural and MEP elements contained within them.
  • CSV Export — Exports detailed element data to an editable CSV format.
  • CSV Import — Re-imports your modified CSV and applies changes directly to the element parameters in the Revit model via an automated transaction.

Usage Workflow

  1. Click Room Wizard in the BIM Digital Design ribbon.
  2. Select whether you want to use Host Rooms or Linked Rooms as the boundary reference.
  3. Review the populated elements in the UI DataGrid.
  4. Click Export and save the generated CSV file.
  5. Open the CSV in Excel, modify the parameters as needed, and save.
  6. Click Import in the Room Wizard UI to load your modified CSV.
  7. Click Update Model to apply the changes to the Revit elements.
Note: The ElementId and Category columns in the CSV are used to map the data back to Revit. Do not modify or delete these columns. Read-only parameters cannot be updated via import.
