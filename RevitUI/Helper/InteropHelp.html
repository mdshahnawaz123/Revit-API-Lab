Interoperability Mission Control

A real-time dashboard that shows element coverage and data health for IFC/COBie readiness.

What It Shows

  • Category Breakdown — Top 15 element categories by count
  • Data Coverage % — Percentage of elements with IfcGuid, Mark, or Comments filled in
  • Progress Bars — Visual indicator of completeness per category

How Coverage Is Calculated

An element is considered "covered" if it has at least one of: IfcGuid, Mark, or Comments parameter filled.

Usage

  1. Click Interop Dashboard in the Model panel
  2. The dashboard scans the model automatically
  3. Click Refresh Model to re-scan after filling in data
