Smart Workset Automator

Bulk-assign elements to their correct worksets based on category mapping. Adapts to your project's unique workset naming convention.

How It Works

  1. The tool scans the model and lists all element categories currently in use
  2. It retrieves all User Worksets from your project
  3. You map each category to a target workset using the dropdown
  4. Click Apply to Model to move all matching elements

Important Notes

Workshared models only — This tool requires the model to be workshared.
"No Change" — Select this to skip a category.
Read-only elements — Elements you don't own will be skipped automatically.

Example Mapping

  • Walls → AR_Structure
  • Ducts → MEP_Ventilation
  • Pipes → MEP_Plumbing
  • Structural Columns → STR_Frame
