Model Purge Plus

Deep cleaning utility that goes beyond standard Revit "Purge Unused" to target hidden clutter and model bloat.

Purge Categories

  • Unused View Templates — Templates not applied to any view
  • Unused View Filters — Filters not used in any view's Visibility/Graphics
  • Imported Line Styles — CAD-imported graphic styles (e.g., ".dwg" styles)
  • Redundant Rooms — Rooms with Area = 0 (Not Enclosed, Not Placed, Redundant)
  • Empty Views — Sections/Elevations with no visible elements
⚠️ Warning: These actions cannot be undone. Always save your model before running a deep clean.

Usage

  1. Click Purge Plus in the QC Panel
  2. Select the categories you want to clean
  3. Click Start Deep Clean
  4. Review the summary report
