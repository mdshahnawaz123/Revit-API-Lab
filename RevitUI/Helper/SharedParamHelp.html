Shared Parameter Manager

Bulk-apply shared parameters from your .txt file to multiple Revit categories in one operation.

How To Use

  1. Browse — Select your shared parameter .txt file
  2. Load — Click Load to read all parameter groups and definitions
  3. Select Parameters — Check the parameters you want to add (use search to filter)
  4. Select Categories — Check the categories to bind to (Select All / Clear All available)
  5. Choose Binding — Instance (per-element) or Type (per-type)
  6. Choose Group — Where the parameter appears in Properties (Identity Data, Dimensions, etc.)
  7. Apply — Click Apply to Project

Important Notes

Already Existing: Parameters that already exist in the project will be skipped automatically.
Instance vs Type: Instance parameters can hold different values per element. Type parameters share values across all instances of the same type.
