Slope Analysis

Visualize surface steepness across Floors, Roofs, and Topography using a heat-map color override in the active view.

Supported Elements

  • Floors — Detects sloped floor surfaces
  • Roofs — Analyzes roof pitch angles
  • Topography — Evaluates terrain surface gradients

Color Legend

Green = Flat (0–5°)
Yellow = Moderate (5–15°)
Orange = Steep (15–30°)
Red = Critical (30°+)

Usage

  1. Open a 3D view
  2. Click Slope Analysis in the Model panel
  3. The tool will apply color overrides to the active view
