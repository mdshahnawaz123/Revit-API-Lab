Dimension Automation

Automatically place dimensions across your model using intelligent detection algorithms.

Modes Available

  • Grid to Grid — Detects parallel grids (vertical and horizontal) and places continuous dimensions along each group with a configurable offset distance.
  • Wall Openings — Scans all walls in the active view, detects parallel faces, and places thickness dimensions automatically.
  • MEP Sleeves — Finds Opening/Sleeve families and dimensions them to the nearest grid line and structural column.
  • Room Boundary — (Coming Soon) Center-to-center room boundary dimensioning.

Offset Distance

The offset value (in mm) controls how far the dimension line is placed from the grid/wall. Default: 1000mm.

Usage

  1. Open a plan view with visible grids/walls
  2. Click Dimension Auto in the Model panel
  3. Select a mode and click Run Automation
