BIM Digital Design – Room 3D Tag Generator

Autodesk Revit Add-in for Automatic 3D Room Labeling

Description

The Room 3D Tag Generator automatically creates 3D text labels for rooms in a Revit model using room names and/or numbers. It enhances model readability, coordination, and presentation in 3D views.

Key Features

  • Automatic 3D room tag generation
  • Uses room name and/or number
  • Places tags at the center of rooms
  • Visible in 3D, plan, and section views
  • Supports large models

Installation

  • Run the BIM Digital Design Setup (.exe)
  • Installer automatically detects Revit versions (2024–2027)
  • Copies required DLL and .addin files to correct locations
  • No manual configuration required

Installed File Locations

C:\ProgramData\Autodesk\Revit\Addins\2024
C:\ProgramData\Autodesk\Revit\Addins\2025
C:\ProgramData\Autodesk\Revit\Addins\2026
C:\ProgramData\Autodesk\Revit\Addins\2027

Usage

  • Open your Revit project
  • Navigate to the BIM Digital Design tab
  • Click "Room 3D Tag"
  • The tool automatically generates tags for all rooms

Uninstallation

  • Use Windows "Add or Remove Programs"
  • All DLL, add-in, and dependency files are automatically removed
  • No manual cleanup required

Limitations

  • Supports Rooms only (Spaces not included)
  • Requires valid room placement

Troubleshooting

  • Ensure rooms are placed in the model
  • Verify required family is loaded
  • Check visibility settings in the view
