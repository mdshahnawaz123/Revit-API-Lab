Room 3D Text Generator

Revit Add-in for Automatic 3D Room Labeling

Description

The Room 3D Text Generator automatically creates 3D model text for all rooms in your Revit project based on room names (and optionally numbers).

This enhances visualization in 3D views and helps with presentation, coordination, and model readability.

General Usage Instructions

  • Open your Revit project.
  • Navigate to the Add-Ins tab.
  • Click Room 3D Text.
  • The tool will automatically:
    • Collect all placed rooms
    • Read room names
    • Place 3D text at the center of each room

Features

  • Automatically generates 3D room labels
  • Uses room name and/or number
  • Places text at room center
  • Visible in 3D, plan, and section views
  • Supports large models

How It Works

  • Collects elements from OST_Rooms
  • Retrieves room location (LocationPoint)
  • Places Generic Model family instance
  • Sets parameter value to Room Name

Installation

Steps:

  • Copy the .addin file to:
C:\ProgramData\Autodesk\Revit\Addins\2024
  • Copy the DLL file to the same folder
  • Restart Revit

Family Requirements

  • Category: Generic Model
  • Contains Model Text element
  • Text parameter linked to "RoomName"

Known Issues

  • Rooms without location are skipped
  • Unplaced rooms are ignored
  • Large models may affect performance

Limitations

  • Supports Rooms only (Spaces not included)
  • Requires valid room boundaries

Troubleshooting

  • If no text appears, check if rooms are placed
  • Ensure family is loaded correctly
  • Verify model text visibility in view settings
