🚀 AutoCAD Batch Export (BDD)

Professional CAD coordination and automated merge pipeline.

Overview

This tool automates the consolidation of exported Revit sheets into professional AutoCAD deliverables. It eliminates manual file manipulation by using an automated merge engine.

Export Modes

  • Multiple Layouts: Each Revit sheet becomes its own Layout tab in a single master DWG. The tool automatically re-centers viewports to match shifted Model Space geometry.
  • Model Space: All sheets are placed side-by-side in Model Space with title blocks, separated by a 100,000 unit gap to avoid overlaps.
  • Single Layout: All sheets are combined into a single master Layout tab.
  • Separate Files: Standard export where each sheet is a standalone DWG file.

How it Works

  1. Select the desired Export Mode.
  2. Pick the destination folder.
  3. Select the sheets/views you wish to export.
  4. Click Export to DWG.

Note: The tool will automatically look for accoreconsole.exe to perform a headless merge in the background. If not found, it will fallback to the full AutoCAD application.

Output

The final combined file will be saved as Combined CAD.dwg in your selected folder.
