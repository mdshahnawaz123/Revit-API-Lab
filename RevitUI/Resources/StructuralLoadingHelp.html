Structural Loading Diagram Tool

Automate the creation of structural loading diagrams in your Revit views. This tool allows you to visualize room types and wall types with color-coded filled regions and detail lines.

Key Features

  • Room Loading: Automatically generates color-coded FilledRegion elements based on room boundaries. Each room type is assigned a unique hatch pattern and color.
  • Wall Loading: Creates DetailLine elements along wall location curves. Line styles are automatically generated and colored based on the wall type name.
  • Intelligent Updates: The tool detects changes in room boundaries or wall lengths. If no changes are detected, existing elements are skipped to maintain performance.
  • Linked Model Support: Process rooms and walls directly from linked models with automatic coordinate transformation.
  • Legend Creation: Generate a visual legend at a picked point to explain the color coding for both rooms and walls.

How to Use

  1. Select Source: Choose between "Host Model" or "Linked Model".
  2. Sync Elements: Click Room Sync or Wall Sync to populate the lists with elements from the active view.
  3. Select Items: Check the boxes for the rooms or walls you want to include in the diagram.
  4. Generate: Click Create Region or Create Line to generate the graphics.
  5. Add Legend: Click Create Legend and pick a point in the view to place the visual key.
Tip: Press F1 while the window is open to return to this help page at any time.
