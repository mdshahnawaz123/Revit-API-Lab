Parameter Filter Tool

The Parameter Filter tool allows users to filter and isolate BIM elements based on selected categories and parameter conditions.

How It Works

  • Select a Category (e.g., Walls)
  • Choose a Parameter (e.g., Workset)
  • Select a Condition (Equals, Contains, etc.)
  • Enter a Value (e.g., Wall)

Buttons

  • Apply Filter: Applies the filter to the model
  • Isolate: Shows only filtered elements
  • Clear: Resets all filter inputs
  • Close: Closes the dialog

Example

Category: Walls
Parameter: Workset
Condition: Equals
Value: Wall

Benefits

  • Quickly locate specific elements
  • Improve model navigation
  • Enhance coordination workflows

Tips

  • Use consistent parameter naming for better results
  • Combine with isolate for focused review
