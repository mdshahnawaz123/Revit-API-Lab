Master Opening Tools

BIM Digital Design - Professional Workflow Solutions

Overview

The Master Opening tools provide an automated workflow for detecting interferences and creating wall, floor, and ceiling openings in your Revit models. Whether you're working with local MEP elements or linked architectural/structural models, these tools streamline the coordination process.

🛠 MEP Opening (Manual/Selection)

Designed for rapid creation of openings based on MEP services (Pipes, Ducts, Cable Trays) within the host model.

  • Select the MEP categories to scan.
  • Define the Tolerance (clearance) around the service.
  • Review detected clashes and click Create Openings.

🚪 Door & Window Opening

Automatically creates rectangular wall openings based on doors and windows found in linked architectural models.

  • Scan linked models for door/window instances.
  • Verify placement in the data grid.
  • Batch create openings that stay updated with link changes.
